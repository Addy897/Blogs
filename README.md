# Blogs

Blogs Project with [SvelteKit](https://kit.svelte.dev/docs/) and [django](https://www.djangoproject.com/)
## Developing

## frontend
```bash
npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```
## backend
```bash
python manage.py runserver
```

## Building

```bash
npm run build
```
